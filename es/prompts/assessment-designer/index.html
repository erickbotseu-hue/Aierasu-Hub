Quick quiz (5-10 questions, auto-graded) - Reflection prompt (short written response) - Practical exercise (apply the concept) - Peer review activity (evaluate someone else's work) - Discussion contribution (demonstrate thinking) 3. **Summative Assessment (final):** | Component | Weight | Description | Rubric | **Option A: Final Project** - Project brief with clear deliverables - Rubric with 4 levels: Exceptional / Proficient / Developing / Beginning - Criteria: content knowledge, application, presentation, originality **Option B: Final Exam** | Section | Question Type | Points | Time | - Section 1: Multiple choice (knowledge) — 30% - Section 2: Short answer (understanding) — 30% - Section 3: Essay/case study (analysis + application) — 40% **Option C: Portfolio** - Collection of best work throughout the course - Training: how to give constructive feedback - Calibration: review sample work together before peer assessing 5. **Self-Assessment Reflection:** | Question | Purpose | - "What did I learn that surprised me?" - 4-6 criteria - Each level has SPECIFIC, OBSERVABLE descriptors - No vague language ("good" or "adequate") - Weight: percentage of total grade per criterion CONSTRAINTS: - Assessments must align with LEARNING OBJECTIVES — don't test what you didn't teach - Use Bloom's Taxonomy to ensure assessments cover multiple cognitive levels (not just recall) - Rubrics must be shared BEFORE the assessment — students should know how they'll be evaluated - Include BOTH knowledge and application — knowing facts isn't the same as using them -
