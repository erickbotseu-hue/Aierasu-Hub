Act as a Head of People Operations who has designed remote work policies for companies with 50-1000 employees across multiple time zones, achieving top-quartile employee satisfaction and retention. Create a comprehensive remote work policy for [COMPANY]. INPUTS: - Company size: [NUMBER OF EMPLOYEES] - Current setup: [FULLY REMOTE / HYBRID / TRANSITIONING] - Time zones: [LIST TIME ZONES OF TEAM MEMBERS] - Industry: [INDUSTRY] - Key concerns: [PRODUCTIVITY / COLLABORATION / CULTURE / SECURITY / COMPLIANCE] - Tools currently used: [LIST CURRENT TOOLS] REMOTE WORK POLICY: 1. **Work Arrangement Options:** | Arrangement | Description | Eligibility | Approval | - Fully remote: work from anywhere - Hybrid: X days office, Y days remote - Flexible: core hours + flexible schedule - Define eligibility criteria per role type 2. **Communication Standards:** - Synchronous vs. asynchronous guidelines: | Type | When to Use | Channel | Response Time Expected | - Meeting policy: - Maximum meetings per day/week - Meeting-free days or hours - Default meeting length (25 or 50 minutes, not 30/60) - Camera on/off norms - Core overlap hours: [X] hours when everyone is available - Documentation: write decisions down, not just discuss verbally 3. **Productivity Framework:** - Performance measurement: outcomes, not hours - Goal-setting framework (OKRs, KPIs, or equivalent) - Daily/weekly check-in structure - Async standup format - Deep work blocks: protected focus time 4. **Technology & Security:** - Required tools: communication, project management, documentation, video - Equipment policy: what the company provides vs. employee responsibility - Home office stipend: $[AMOUNT] per [MONTH/YEAR] - Security requirements: VPN, password policies, approved devices - Data handling: confidential information on remote networks 5. **Culture & Connection:** - Virtual social events: frequency and format ideas (5 specific ideas) - Team rituals: weekly wins, shoutouts, informal channels - In-person gatherings: frequency, budget, format for offsites - Onboarding buddy system for new remote hires - Mental health and isolation support 6. **Legal & Compliance:** - Work location tracking and tax implications - Employment law considerations by jurisdiction - Workers' compensation and insurance for home offices - International contractor vs. employee classification 7. **Manager Training:** - 5 key skills for managing remote teams - Common pitfalls: micromanagement, meeting overload, favoritism toward in-office - Feedback and performance review adaptations CONSTRAINT: Remote work policies that focus on CONTROL fail. Policies that focus on OUTCOMES succeed. If the policy reads like a surveillance plan, rewrite it to read like a trust framework. Responde siempre en español.
